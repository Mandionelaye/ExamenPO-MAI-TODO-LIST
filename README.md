# ExamenPO-MAI-TODO-LIST

Une application web simple et intuitive permettant à un utilisateur de gérer efficacement sa liste de tâches quotidiennes. Ce projet a été réalisé dans le cadre d'un examen de programmation.

## 📋 Description

L'application **ExamenPO-MAI-TODO-LIST** a pour objectif de fournir une interface utilisateur claire pour :
*   Ajouter de nouvelles tâches.
*   Lister les tâches existantes.
*   Marquer des tâches comme terminées.
*   Supprimer des tâches.

Il s'agit d'un projet de démonstration construit avec **Java** et **Spring Boot**, utilisant **Maven** pour la gestion des dépendances.

## 🛠️ Technologies Utilisées

*   **Java** - Langage de programmation principal.
*   **Spring Boot** - Framework pour la création de l'application web.
*   **Maven** - Outil de gestion de dépendances et de construction du projet.
*   **Thymeleaf** - Moteur de template pour l'interface utilisateur côté serveur.
*   **Base de données MySQL** - Pour le stockage persistant des tâches.
